The World Wide Web is a collection of HTML (Hypertext Markup Language) Documents by which HTML "glues" Documents, Images, videos together and Hyperlinks or "links" makes up the web

Major Pieces

- You, your computer, internet, HTTP and Servers make up the major pieces
- Browsers such as Internet Explorer, Chrome, Firefox use HTTP to communicate with the servers.

Key Definitions

- HTTP = Main protocol of the web.
- servers = hosts the files that make up the web
- Internet = Largest computer network
- Browser = program on your computer to display the files on the web.
- HTML = (Hypertext Markup Languange) is the heart of the web; takes text content and by markup to what it looks like on the web

HTML Markup

- Start Tag <>content<> close Tag . From start to close is an Element.
- Element is everything within a set of opening and closing tags.
- Key Tags <> b= bold, em= italic
- Void Tags do not close <> br= break
- HTML Attributes (ATTR) Always a value.
- Anchor <> a includes: attribute; value; content; closing tag
- <> img for images to be embedded - <>p paragraph

Inline vs. Block

- Block has a height and width and can be managed with variations/distinctions around the text. Inludes <>p
- Inline impacts the normal line items. Word within a sentence."nothing fancy".

- Inline Inclues <> b, em, img, a, span, strong
- Block includes <>div, <>p, <>form
